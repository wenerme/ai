# Security

Find, review, and remediate vulnerabilities in authorized code.

Use the Codex Security plugin to assess code you have permission to review and
turn evidence into bounded remediation work. Start with read-only detection,
review findings, and approve changes deliberately.

## Find, validate, and fix vulnerabilities

Scan an authorized repository or code change, validate reviewed findings, and
prepare focused fixes with evidence for approval.
