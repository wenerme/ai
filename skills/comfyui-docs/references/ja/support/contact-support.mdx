サポートチャンネルを通じて ComfyUI のヘルプを取得し、適切なリポジトリで問題を報告してください。

## サポートチャンネル

  
    サポートチームからヘルプを取得できます。
  

## 問題を報告する

バグおよび問題は、対応する GitHub リポジトリで報告してください：

  
    ComfyUI_frontend の UI 関連問題を報告できます。
  

  
    コア機能に関する問題を報告できます。
  

  
    デスクトップ版に関する問題を報告できます。
