Wan2.7 は、Alibaba の最新の動画生成モデルで、パートナーノードを通じて ComfyUI で利用可能になりました。バージョン 2.6 からの包括的なアップグレードであり、画質、音声、モーションダイナミクス、スタイライゼーション、一貫性が大幅に向上しています。

今回のリリースでは、完全にアップグレードされたマルチモーダル動画パイプラインがノードグラフに直接統合され、テキスト、画像、音声、動画の入力に対応し、5 つのタスクタイプをサポートします。

## 主な機能

- **画像から動画**：先頭フレーム、先頭＋最終フレーム、音声駆動による生成
- **テキストから動画**：テキストプロンプトのみで生成可能。オプションで音声入力やマルチショットナレーションにも対応
- **動画の続き**：テキストプロンプトで誘導しながら、既存のクリップに新しいコンテンツを追加
- **参照から動画**：被写体の視覚的外観と声色の両方を参照可能。最大 5 名の実在人物入力とマルチキャラクターインタラクションをサポート
- **動画編集**：テキストプロンプト、参照画像、またはスタイル転送による動画の編集・複製

## ハイライト

- マルチキャラクターシーン向けに**最大 5 名の実在人物画像入力**をサポート
- 一貫した音声・視覚アイデンティティのための声色参照
- 3x3 グリッドベースの画像生成
- Wan2.6 と比較して、モーションダイナミクス、スタイライゼーション、一貫性が大幅に向上

## Wan2.7 画像から動画

画像入力から動画を生成します。先頭フレーム、先頭＋最終フレーム、音声駆動生成モードに対応しています。

  
    Wan2.7 画像から動画ワークフローファイルを取得します。
  
  
    Comfy Cloud 上で画像から動画ワークフローをすぐにお試しいただけます。
  

## Wan2.7 テキストから動画

テキストプロンプトのみから動画を生成します。オプションで音声入力やマルチショットナレーションを追加し、より豊かなストーリーテリングが可能です。

  
    Wan2.7 テキストから動画ワークフローファイルを取得します。
  
  
    Comfy Cloud 上でテキストから動画ワークフローをすぐにお試しいただけます。
  

## Wan2.7 参照から動画

被写体の視覚的外観の参照画像と、オプションの声色参照を使用します。マルチキャラクターインタラクションシーン向けに、最大 5 名の実在人物入力をサポートしています。

  
    Wan2.7 参照から動画ワークフローファイルを取得します。
  
  
    Comfy Cloud 上で参照から動画ワークフローをすぐにお試しいただけます。
  

## Wan2.7 動画編集

テキストプロンプト、参照画像、またはスタイル転送を使用して、既存の動画を編集・複製します。

  
    Wan2.7 動画編集ワークフローファイルを取得します。
  
  
    Comfy Cloud 上で動画編集ワークフローをすぐにお試しいただけます。
