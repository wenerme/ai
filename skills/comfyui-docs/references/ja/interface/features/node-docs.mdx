ノードドキュメント機能を使用すると、ワークフロー内の任意のノードのドキュメントにすばやくアクセスできます。ノードの機能、パラメータ、使用例に関する詳細情報を、ComfyUI 内で直接確認できます。

## ノードドキュメントの表示方法

[ノードドキュメント]

1. **ノードを選択:**  
   ワークフロー内の任意のノードをクリックして選択します。

2. **ノードドキュメントを開く:**  
   選択ツールボックス内のノード情報アイコンをクリックして、ノードドキュメントページを開きます。

3. **ノードドキュメントを終了:**  
   ノードドキュメントを終了するには、左上隅の矢印アイコンをクリックして終了します。

ノードライブラリからノードドキュメントにアクセスすることもできます。ノードにマウスカーソルを合わせ、「?」アイコンをクリックしてノードドキュメントを開きます。

[ノードドキュメント]

## 組み込みノードのドキュメント

ComfyUI の組み込みノードのドキュメントは、[embedded-docs リポジトリ](https://github.com/Comfy-Org/embedded-docs) で管理されています。

**組み込みノードのドキュメントへの貢献:**
- ユーザーは組み込みノードのドキュメントの改善や追加に貢献できます
- [embedded-docs リポジトリ](https://github.com/Comfy-Org/embedded-docs) にプルリクエストを送信します
- ComfyUI コミュニティ全体のドキュメント品質の向上に役立ちます

## カスタムノードのドキュメント

カスタムノードも独自のドキュメントを含めることができます。カスタムノードの作者がドキュメントを提供している場合、そのノードを選択するとノードドキュメントパネルに自動的に表示されます。
