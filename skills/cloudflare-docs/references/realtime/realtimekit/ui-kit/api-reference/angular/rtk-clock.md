# rtk-clock

Shows the time elapsed in a meeting.

## Properties

| Property | Type     | Required | Default         | Description    |
| -------- | -------- | -------- | --------------- | -------------- |
| iconPack | IconPack | ❌        | defaultIconPack | Icon pack      |
| meeting  | Meeting  | ✅        | \-              | Meeting object |
| size     | Size     | ✅        | \-              | Size           |

## Usage Examples

### Basic Usage

```

<!-- component.html -->

<rtk-clock></rtk-clock>


```

### With Properties

```

<!-- component.html -->

<rtk-clock

 [meeting]="meeting"

 size="md">

</rtk-clock>


```
