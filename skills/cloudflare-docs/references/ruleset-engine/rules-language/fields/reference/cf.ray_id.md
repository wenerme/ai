#  cf.ray\_id 

`cf.ray_id` ` String ` 

The Ray ID of the current request.

A [Ray ID](https://developers.cloudflare.com/fundamentals/reference/cloudflare-ray-id/) is an identifier given to every request that goes through Cloudflare.
