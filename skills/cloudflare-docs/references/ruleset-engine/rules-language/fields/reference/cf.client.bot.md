#  cf.client.bot 

`cf.client.bot` ` Boolean ` 

Indicates whether the request originated from a known good bot or crawler.

Provides the same information as [cf.bot\_management.verified\_bot](https://developers.cloudflare.com/ruleset-engine/rules-language/fields/reference/cf.bot%5Fmanagement.verified%5Fbot/).
