# Get started

To create and manage tunnels, you will need to install and authenticate `cloudflared` on your origin server. `cloudflared` is what connects your server to Cloudflare's global network.

* [ Create a tunnel (dashboard) ](https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/get-started/create-remote-tunnel/)
* [ Create a tunnel (API) ](https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/get-started/create-remote-tunnel-api/)
* [ Useful terms ](https://developers.cloudflare.com/cloudflare-one/networks/connectors/cloudflare-tunnel/get-started/tunnel-useful-terms/)
