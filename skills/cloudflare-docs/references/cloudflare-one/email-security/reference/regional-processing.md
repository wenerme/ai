# Regional processing

Email security uses Cloudflare's [Data Localization Suite (DLS)](https://developers.cloudflare.com/data-localization/) to allow you to control where your emails are processed.

Note

Regional processing is only available for customers who deploy Email security via MX/Inline or BCC/Journaling.

Currently, you can process emails in the following regions:

* US
* Germany
* India
* Australia
