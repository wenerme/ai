# Nano Banana (Gemini 2.5 Flash) Prompt Guide

Nano Banana 是 Google Gemini 2.5 Flash 的图像生成能力代号。与 SD/DALL-E 不同，它原生支持**图像编辑**（image-to-image）和**中文 prompt**，能理解上下文、光照物理和 3D 空间关系。

## 核心特点（vs SD/DALL-E）

| 特点 | Nano Banana | SD/DALL-E |
|------|-------------|-----------|
| 中文 prompt | 原生支持，效果好 | 英文优先 |
| 图像编辑 | 原生支持（传入参考图） | 需要 img2img/inpaint |
| 3D 空间理解 | 强（遮挡、透视、光影） | 一般 |
| 风格一致性 | 强（单张参考图推断风格） | 需要 LoRA/ControlNet |
| 文字渲染 | 中英文都较好 | 通常较差 |
| Negative prompt | 不需要 | SD 强依赖 |

## Prompt 原则

1. **直接用中文描述** — 不需要翻译成英文，中文效果往往更好
2. **描述具体场景而非关键词堆叠** — 用完整句子描述你想要的画面
3. **明确区分风格区域** — 例如"人物用 Q 版 3D 风格，环境用写实风格"
4. **利用参考图** — 传入照片 + 文字指令，比纯文字效果好很多
5. **不需要 negative prompt** — 直接说想要什么，不用说不要什么

## 风格模板（从 100 个实战案例总结）

### 1. 3D Q 版 / Chibi（最热门）

```
将照片里的人物转换成 Q 版 3D 卡通人物，[场景描述]。
人物采用 Q 版 3D 风格，其他环境采用真实写实风格。
```

变体：
- **Q 版情侣场景**：求婚/婚礼/水晶球/珠宝盒摆件
- **Q 版表情包**：生成 9 宫格表情（开心/生气/惊讶/哭泣/得意/害羞/无语/搞怪/比心）
- **Q 版钥匙串**：特写照片，可爱钥匙串造型，柔软橡胶材质，粗黑描边
- **Q 版拍立得出框**：角色从拍立得照片中走出，突破二维边框

### 2. 产品广告 / 品牌设计

```
一个柔和的 3D 卡通风格 [品牌产品] 雕塑，
由光滑的粘土般纹理和鲜艳的柔和色彩制成，
放置在简约的等距场景中，构图简洁，光线柔和，
产品徽标和口号清晰显示在下方。
```

变体：
- **极简 3D 插画**：使用 JSON 结构化 prompt 描述 art_style_profile
- **未来主义 Logo 交易卡**：全息效果、渐变色、金属质感
- **品牌键帽**：机械键盘键帽造型的品牌 Logo

### 3. 复古 / 怀旧风格

```
复古宣传海报风格，突出中文文字，
背景为红黄放射状图案，中心位置有 [主体描述]，
以精致复古风格绘制。
```

变体：
- **CRT 电脑启动屏幕**：复古绿色磷光显示器、DOS 风格文字
- **35mm 胶片风格**：胶片颗粒感、柔和色彩、梦幻氛围
- **PS2 游戏封面**：经典 PS2 包装盒风格

### 4. 特效合成

```
[描述特效类型] 效果，[主体描述]。
```

- **双重曝光**：人像 + 风景叠加
- **透视 3D 出屏**：角色从画框/手机/照片中突出
- **磨砂玻璃剪影**：虚实对比，背光效果
- **半透明玻璃质感**：将实物转换为通透玻璃材质

### 5. 风格转换

- **吉卜力风格**：宫崎骏动画风格重绘
- **乐高风格**：将场景/人物转换为乐高积木
- **像素风格**：8-bit 像素图标，复古调色板
- **纸艺风格**：剪纸/折纸立体效果
- **体素风格**：Minecraft 风格 3D 方块

### 6. 实用设计模板

- **小红书封面**：社交媒体封面设计
- **手绘信息图卡片**：手绘风格数据可视化
- **代码风格名片**：程序员名片，代码编辑器风格
- **扁平贴纸设计**：可爱扁平风贴纸集合
- **RPG 角色卡片**：游戏角色属性卡

### 7. 创意场景

- **水晶球故事场景**：微缩场景在水晶球中
- **微型立体场景**：等距微缩世界
- **蒸汽朋克机械**：机械零件 + 复古铜色
- **发光线条解剖图**：X 光风格透视

## 进阶技巧

### 结构化 Prompt（JSON 格式）

Nano Banana 支持 JSON 结构化 prompt，适合复杂场景：

```json
{
  "scene": "一个等距的微型城市街角",
  "art_style_profile": {
    "style_name": "Minimalist 3D Illustration",
    "color_palette": ["#FF6B6B", "#4ECDC4", "#FFE66D"],
    "rendering_style": {
      "technique": "3D render with simplified, low-poly aesthetics",
      "shadow_style": "Subtle and diffused"
    }
  },
  "text_overlay": {
    "content": "Hello World",
    "font_style": "Minimalistic, sans-serif"
  }
}
```

### 混合风格指令

Nano Banana 擅长在同一画面中混合不同风格：

```
除了人物采用 Q 版 3D 人物风格，其他环境采用真实写实风格。
```

这种"局部风格化 + 整体写实"是 Nano Banana 的独特优势。

### 利用 Emoji 作为主体

直接用 emoji 描述想要的主体，Nano Banana 理解力很强：

```
创建一个极简主义的 8 位像素风格的 [🍔] 标志，
居中放置在纯白背景上，使用有限的复古调色板。
```
