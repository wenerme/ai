## Layout Details

### bento-grid (DEFAULT)
- Grid of rectangular cells with varied sizes
- Hero cell for main point, supporting cells around
- Clear cell boundaries, consistent padding
- Recommended: `craft-handmade`, `corporate-memphis`, `pixel-art`

### linear-progression
- Sequential steps connected by arrows or path
- Horizontal (timeline feel) or vertical (process feel)
- Each step: icon + label + brief description
- Recommended: `craft-handmade`, `ikea-manual`, `corporate-memphis`

### binary-comparison
- Two-column split with clear visual divider
- Each side: icon + key points + visual representation
- Color-coded sides for instant recognition
- Recommended: `corporate-memphis`, `bold-graphic`

### hierarchical-layers
- Pyramid or stacked horizontal bands
- Top layer = highest priority/smallest group
- Bottom layer = foundation/largest group
- Recommended: `craft-handmade`, `technical-schematic`

### hub-spoke
- Central circle for core concept
- Radiating spokes to related items
- Equal weight for connected elements
- Recommended: `craft-handmade`, `corporate-memphis`

### funnel
- Progressive narrowing from top to bottom
- Each stage labeled with counts/percentages
- Color gradient from wide to narrow
- Recommended: `corporate-memphis`, `craft-handmade`

### dashboard
- Card-based layout with prominent numbers
- KPI cards, mini charts, status indicators
- Grid alignment, consistent spacing
- Recommended: `corporate-memphis`, `technical-schematic`

### dense-modules
- Tightly packed content blocks
- Maximizes information per area
- For data-rich guides and reference sheets
- Recommended: `morandi-journal`, `pop-laboratory`, `retro-pop-grid`
