## Key Features

- **Native multimodal**: Text, image, audio, video input
- **Ultra-long context**: Up to 2M tokens (Gemini 2.0 Pro)
- **Thinking mode**: Available on Gemini 2.5 Flash/Pro
- **Grounding**: Google Search integration
- **Code execution**: Built-in Python sandbox
- **Structured output**: JSON schema enforcement
